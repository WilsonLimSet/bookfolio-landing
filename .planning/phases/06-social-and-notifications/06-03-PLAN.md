Web app leaderboard (leaderboard/page.tsx):
- Data: fetch liked books (tier=liked, limit 200) and active users (user_books user_id, limit 1000)
- Top Fiction (10): aggregate liked books by open_library_key, avg score, majority-rules category (fiction wins ties), sort by avg score desc
- Top Nonfiction (10): same but nonfiction category
- Most Active (10): count user_books per user, sort by count desc, fetch profiles
- Display: rank badge, cover/avatar, title/username, avg score or book count

Leaderboard aggregation algorithm:
1. Group liked books by open_library_key
2. For each group: calculate avg score, count ratings, determine category by majority vote (fiction wins ties)
3. Split into fiction and nonfiction
4. Sort each by avg score descending, take top 10
5. For active users: count user_books per user_id, sort by count desc, take top 10

struct LeaderboardBook: Identifiable {

Methods:

a. `static func fetchLeaderboardData() async -> (fiction: [LeaderboardBook], nonfiction: [LeaderboardBook], activeUsers: [LeaderboardUser])`:

   - Fetch in parallel (async let):
     1. Liked books: `supabase.from("user_books").select("open_library_key, title, author, cover_url, score, category").eq("tier", "liked").order("score", ascending: false).limit(200)`
     2. All user books (for active count): `supabase.from("user_books").select("user_id").limit(1000)`

   - Use private Decodable row types:
     ```swift
     private struct LikedBookRow: Decodable {
         let openLibraryKey: String
         let title: String
         let author: String?
         let coverUrl: String?
         let score: Double
         let category: String
     }
     private struct UserIdRow: Decodable {
         let userId: UUID
     }
     ```

   - Aggregate liked books by openLibraryKey:
     - Group into dictionary: [openLibraryKey: [LikedBookRow]]
     - For each group:
       - avgScore = sum(scores) / count
       - fictionVotes = count where category == "fiction"
       - nonfictionVotes = count where category == "nonfiction"
       - category = fictionVotes >= nonfictionVotes ? "fiction" : "nonfiction" (fiction wins ties)
       - Take title, author, coverUrl from first entry
     - Split into fiction and nonfiction arrays
     - Sort each by avgScore descending
     - Take top 10 of each
     - Map to LeaderboardBook

   - Count active users:
     - Group user_books by userId → dictionary [UUID: Int]
     - Sort by count descending, take top 10
     - Fetch profiles: `supabase.from("profiles").select("id, username, avatar_url").in("id", topUserIds)`
     - Map to LeaderboardUser with bookCount

   - Return tuple

IMPORTANT: Use try? for all queries — leaderboard is non-critical, show empty on failure.
IMPORTANT: Fiction wins ties in category voting (same as web app).
IMPORTANT: CodingKeys for snake_case mapping on private row types.
