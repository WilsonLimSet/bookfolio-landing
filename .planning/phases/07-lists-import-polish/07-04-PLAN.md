Privacy considerations for this app:
- Camera/Photo Library: avatar upload (PhotosPicker)
- Push Notifications: notification permission
- Network: all Supabase API calls, OpenLibrary API calls
- File access: Goodreads CSV import (fileImporter)
- No tracking, no location, no contacts, no health data

Apple requirements:
- PrivacyInfo.xcprivacy required for apps using certain APIs
- NSPhotoLibraryUsageDescription for PhotosPicker
- NSCameraUsageDescription if camera access (we don't use camera directly)
- Privacy nutrition labels in App Store Connect (manual step)
