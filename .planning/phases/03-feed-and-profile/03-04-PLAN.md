2. CREATE ReadBooksView.swift — displays user's ranked books:

Properties:
- `let userId: UUID`
- `@State private var fictionBooks: [UserBook] = []`
- `@State private var nonfictionBooks: [UserBook] = []`
- `@State private var isLoading = true`

Load on appear:
- Fetch user_books WHERE user_id = userId, ordered by rank_position
- Split into fiction/nonfiction arrays by category

Layout:
- NavigationTitle: "Read"
- ScrollView with two sections:
  - "Fiction" header + LazyVStack of BookRowView (with rank position and score)
  - "Nonfiction" header + LazyVStack of BookRowView
- Empty state: "No books ranked yet" with book icon

Tap on book → navigate to .bookDetail(bookKey:) via NavigationLink or programmatic push.

3. CREATE CurrentlyReadingView.swift:

Properties:
- `let userId: UUID`
- `@State private var books: [CurrentlyReading] = []`
- `@State private var isLoading = true`

Load: `from("currently_reading").select().eq("user_id", ...).order("started_at", ascending: false)` decoded as [CurrentlyReading]

Layout:
- NavigationTitle: "Currently Reading"
- LazyVGrid (2 columns, spacing 16):
  - Each cell: cover image (rounded), title below, author below
- Empty state: "Not reading anything right now" with book.closed icon

4. CREATE WantToReadView.swift:

Properties:
- `let userId: UUID`
- `@State private var books: [WantToRead] = []`
- `@State private var isLoading = true`

Load: `from("want_to_read").select().eq("user_id", ...)` decoded as [WantToRead]

Layout:
- NavigationTitle: "Want to Read"
- LazyVGrid (2-3 columns):
  - Each cell: cover image (rounded), title, author
- Empty state: "Your reading wishlist is empty" with bookmark icon

IMPORTANT: AsyncImage for book covers should use the coverUrl field. Handle nil coverUrl with a placeholder (book.fill SF Symbol).
IMPORTANT: For LazyVGrid, use GridItem(.flexible()) for responsive columns.
