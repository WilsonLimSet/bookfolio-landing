Add these static methods to the existing ProfileService:

1. `static func updateProfile(_ update: ProfileUpdate, userId: UUID) async throws`:
   - `supabase.from("profiles").update(update).eq("id", value: userId.uuidString).execute()`

2. `static func uploadAvatar(data: Data, userId: UUID) async throws -> String`:
   - Path: "\(userId.uuidString)/avatar.jpg"
   - Upload: `try await supabase.storage.from("avatars").upload(path: path, file: data, options: .init(contentType: "image/jpeg", upsert: true))`
   - Get public URL: `try supabase.storage.from("avatars").getPublicURL(path: path).absoluteString`
   - Return the public URL string

3. `static func fetchUserBooks(userId: UUID) async throws -> [UserBook]`:
   - `from("user_books").select().eq("user_id", ...).order("score", ascending: false)` decoded as [UserBook]
   - For favorite book picker — returns all ranked books sorted by score desc

4. `static func updateFavorites(_ favorites: [FavoriteBook], userId: UUID) async throws`:
   - Delete existing: `from("favorite_books").delete().eq("user_id", ...)`
   - Insert new: for each favorite at index i, insert with position = i + 1
   - Use a loop with individual inserts (simpler than batch with supabase-swift)

IMPORTANT: For avatar upload, supabase-swift storage API uses `upload(path:file:options:)` where file is Data.
IMPORTANT: ProfileUpdate is already defined in Profile.swift as an Encodable struct with optional fields.
IMPORTANT: The upsert option in FileOptions ensures re-uploads overwrite the existing avatar.
